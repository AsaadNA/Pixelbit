This was made as a Project for 2nd Semester course ( Object Oriented Programming [OOP] )

Tool is made for creating pixel art

Features :

- Drawing
- Erase Tool
- Color chooser
- Saving Files
- Loading Files
- Bucket Tool
- Toolbar with toggles & buttons
- Select tool (for erasing)
- Export Art with Transparent BG

Instructions: 

- Use left / right click for filling a pixel
- Use shift + drag mouse for filling multiple pixels
